# 📝 BlogApp

A modern **blog application** built with React and Vite. Create, edit, and share blog posts with a clean and responsive UI.

---

## ✨ Features

- ✍️ **Create & Edit Posts** — Write and manage your blog posts
- 📱 **Responsive Design** — Works seamlessly on all devices
- ⚡ **Fast Performance** — Powered by Vite for lightning-fast builds
- 🎨 **Clean UI** — Modern and intuitive user interface

## 🚀 Getting Started

### Prerequisites
- Node.js 18+

### Installation

```bash
# Clone the repository
git clone https://github.com/DEVRAJ-20/BlogApp.git
cd BlogApp

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`
